# LINE_crypto2.0-main
本程式為一個以 Python 開發的加密貨幣市場監控與訊號通知系統，透過串接 Binance API 即時取得多幣種歷史與最新 K 線資料，並使用 TA-Lib 計算 RSI 技術指標，以判斷市場是否處於超買或超賣狀態。  系統會定時（每小時整點）自動掃描多個交易對，當 RSI 達到預設條件時產生交易訊號（買入或賣出），並整合 LINE Notify API 即時推送通知，讓使用者能快速掌握市場機會。為確保資料即時性，程式會持續更新本地 K 線資料並進行計算。  整體架構結合資料抓取、技術分析、訊號判斷與即時通知，具備自動化監控與實務應用能力，可作為交易策略開發與即時交易系統的基礎。
